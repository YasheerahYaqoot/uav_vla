# Vision-Language-Action Model for Generalist Multicopter Missions

---

MSc Thesis 2025

This repository contains the **Vision-Language-Action (VLA) framework for autonomous mission execution of drones**.  
The framework enables UAVs to interpret **natural language instructions**, identify mission targets from **visual data**, and generate executable flight plans for aerial robots.

The proposed system integrates **Vision-Language Models (VLMs)** and **Large Language Models (LLMs)** to bridge human instructions and autonomous UAV actions.

The framework has been validated in two ways:

- **Benchmark Evaluation**  
  The VLA framework is evaluated on a **custom benchmark dataset** designed for aerial mission generation tasks.  
  The complete implementation and experimental details are available in ```vla_framework/```.

- **Real-World Experiments**  
The framework was also validated through **controlled real-world field experiments with drones**, demonstrating the feasibility of autonomous mission execution in practical environments.  
The implementation and experimental setup can be found in ```implementation/```/
